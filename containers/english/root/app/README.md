# README #

### How do I get set up? ###

* to build app:
```
npm run-script build
```
* to start app:
```
PORT=5000 npm start
```
* to run app (build and star after it with one command):
```
PORT=5000 npm run-script run
```

* deploy docker container:
```
bash -x ./deploy_english_remotely user@your-server.com
```
